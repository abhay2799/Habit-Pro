Habit Pro is a modern, intuitive, and feature-rich Android application designed to help you build, track, and maintain your daily habits effectively. Built with cutting-edge Android technologies and powered by the Gemini API, it provides a seamless user experience to boost your productivity.

## ✨ Features

*   **Daily Habit Tracking:** Easily log your daily habits with a single tap.
*   **Progress Monitoring:** Visualize your consistency over time with detailed analytics and charts.
*   **Smart Reminders:** Never miss a habit with customizable notifications.
*   **Modern UI/UX:** A clean, beautiful interface built entirely with Jetpack Compose.
*   **AI-Powered Insights:** Integrated with the Gemini API to provide intelligent suggestions.

## 🛠️ Tech Stack

*   **Language:** [Kotlin](https://kotlinlang.org/)
*   **UI Toolkit:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
*   **Architecture:** MVVM (Model-View-ViewModel)
*   **Local Database:** Room
*   **Asynchronous Programming:** Kotlin Coroutines & Flow
*   **AI Integration:** Gemini API

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

*   [Android Studio](https://developer.android.com/studio) (Latest Version Recommended)
*   A Gemini API Key (Get one from [Google AI Studio](https://aistudio.google.com/))

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/abhay2799/Habit-Pro.git
    ```
2.  **Open the project** in Android Studio.
3.  **Sync the project** with Gradle files to download dependencies.
4.  **Set up your Environment Variables**
    Create a file named `.env` in the root directory of the project and add your Gemini API key:
    ```properties
    GEMINI_API_KEY=your_api_key_here
    ```
5.  **Run the app** on an emulator or physical device.
